# Website Institucional – Psicologia

Projeto de website institucional desenvolvido como parte do meu portfólio em front-end. A proposta foi criar um site moderno, leve e profissional para uma psicóloga clínica, com foco em experiência do usuário, SEO e performance.

O objetivo foi simular um cenário real de mercado, pensando não apenas no visual, mas também na estrutura técnica e na conversão.

# Sobre o desenvolvimento

O site foi construído com HTML5, CSS3 e JavaScript Vanilla, sem uso de frameworks, priorizando organização, controle da estrutura e melhor desempenho.

A marcação é semântica e bem estruturada, favorecendo acessibilidade e indexação nos mecanismos de busca. O layout foi desenvolvido com abordagem mobile-first, garantindo boa experiência em celulares, tablets e desktop.

Performance e boas práticas

Durante o desenvolvimento, foquei em manter o projeto leve e otimizado. Evitei bibliotecas desnecessárias, organizei os arquivos de forma simples e utilizei imagens otimizadas para melhorar o carregamento.

Também implementei boas práticas recomendadas por ferramentas como Lighthouse, como definição de dimensões de imagens, código enxuto e estrutura organizada.

# SEO e estrutura

O projeto inclui:

Hierarquia correta de títulos
Meta tags otimizadas
Textos alternativos em imagens
Dados estruturados em JSON-LD (incluindo FAQ)

A ideia foi pensar no SEO desde a base, e não como algo adicionado depois.

# Experiência do usuário

O design segue uma linha minimalista, com tons de verde claro, bastante espaço em branco e tipografia limpa. A navegação é simples e intuitiva.

No mobile, implementei menu hambúrguer lateral e botão flutuante de WhatsApp como principal ponto de contato.

# Uso de Inteligência Artificial

Utilizei IA como apoio para revisar estrutura, melhorar organização do código e refinar decisões de SEO e layout. A ferramenta foi usada como suporte de produtividade, mantendo sempre ajustes e validações técnicas manuais.

Organização de código
Estrutura semântica bem aplicada
Responsividade
